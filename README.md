# Vite + Sass Project Template

This is a basic template for starting a web project using Vite and Sass. It provides a minimal setup to quickly begin developing with these technologies.

## Features

*   **Vite:**  A fast build tool that significantly improves development speed and performance.
*   **Sass:** A CSS preprocessor that adds power and elegance to your stylesheets.
*   **Clean Structure:** A simple and organized project structure to keep your code manageable.
* **ES Modules:** Uses ES modules for modern JavaScript development.

## Prerequisites

Before you can run this project, you need to have **Node.js** installed on your system. It's recommended to use at least the **LTS (Long-Term Support)** version of Node.js to ensure compatibility and stability.

You can download Node.js from the official website: [https://nodejs.org/](https://nodejs.org/)

## Getting Started

1.  **Clone the repository:**

    ```
    git clone https://github.com/arnaunin/Vite-Sass-project-template.git
    ```
    
2.  **Navigate to the project directory:**

    ```
    cd Vite-Sass-project-template
    ```
    ```
    cd template
    ```

3.  **Install dependencies:**

    ```
    npm install
    ```
    ```
    npm install -D sass
    ```

5.  **Start the development server:**

    ```
    npm run dev
    ```

    This will start a local development server, usually at `http://localhost:5173/`. The server will automatically reload when you make changes to your files.
